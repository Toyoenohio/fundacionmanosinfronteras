# Manos Sin Fronteras — Landing Page

Landing page para la Fundación Manos Sin Fronteras. Un solo archivo HTML con Tailwind CSS CDN y JavaScript vanilla. Sin dependencias de build, sin framework.

## Estructura

```
├── index.html          ← Todo el sitio (HTML + CSS + JS inline)
└── images/
    ├── hero-bg.jpg
    ├── logo_fundacion.jpeg
    └── quieres-somos.jpg
```

## Secciones

1. **Hero** — Imagen de fondo con parallax + 2 CTA cards
2. **Quiénes Somos** — Misión, cita, junta directiva
3. **Monitoreo** — Contadores animados con datos de la emergencia
4. **Ubicación** — Mapa placeholder + lista de comunidades
5. **Pilares de Acción** — 3 cards (corto/mediano plazo)
6. **Formulario** — Tabs: Suministros / Voluntariado / Fondos
7. **Transparencia** — 3 pilares de gobernanza
8. **Footer** — Navegación, contacto, CTA

## Deploy

Copiar la carpeta a cualquier hosting estático (GitHub Pages, Netlify, Cloudflare Pages, etc.).
